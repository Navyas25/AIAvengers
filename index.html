// Pipeline

How it works

A clean 4-step NLP pipeline from raw document to ranked output.

01

Upload

Drop JD + resumes into the UI or call POST /upload. Accepts PDF, DOCX, TXT.

02

Parse

Text extraction, tokenization, and entity recognition. Handles messy formats gracefully.

03

Match

JD requirements mapped to candidate profiles via TF-IDF + (optional) embeddings.

04

Rank

Weighted scores computed. Results returned as ranked JSON — ready for your HR team.

// API Reference

Developer Endpoints

Simple REST API — drop into any backend or call from the frontend.

POST /upload 
# Upload resumes + JD
POST /api/upload
Content-Type: multipart/form-data

# Body params:
jd_text: "Senior Engineer, 5yr Python..."
resumes[]: [file1.pdf, file2.docx]

# Response:
{
  "session_id": "abc123",
  "files_received": 5,
  "status": "queued"
}
GET /rank 
# Fetch ranked results
GET /api/rank?session=abc123

# Response:
{
  "ranked": [
    {
      "rank": 1,
      "name": "Priya Sharma",
      "score": 94,
      "skills": ["Python", "NLP"],
      "match_pct": 91
    }
  ]
}
